# Q2 Question Creator

一个 Claude Code skill，用于生成 Q2 类型的计算机专业选择题。

## 什么是 Q2 题目

Q2 题目采用**隐式主条件 + 显式特殊条件**的双层结构。题目包含一个隐式的主条件（如"该选项是一棵哈夫曼树"）和 3-5 条显式特殊条件。选项分布在"同时满足两者""仅满足主条件""仅满足特殊条件""两者都不满足"四个象限中，利用条件之间的独立性构造推理陷阱。

核心机制：所有列出选项均不满足"主条件 + 全部特殊条件"，正确答案始终是"以上都不符合"。

## 安装

```bash
claude skills install <github-username>/question-creator-q2
```

## 使用

在 Claude Code 中输入：

```
请针对哈夫曼树出一道Q2类型的题
```

或者

```
出3道Q2类型的题目，知识点分别是红黑树、Dijkstra、前缀编码
```

Skill 会自动生成包含题干、选项、图建议和解析的 Word 文档。
